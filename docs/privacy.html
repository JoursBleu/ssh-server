隐私政策

最后更新：2026 年 3 月 14 日

简要说明：SSH Server 不收集、不存储、不传输任何用户数据。应用完全在本地运行，不联网，不包含任何分析 SDK 或广告 SDK。

1. 应用简介

SSH Server（以下简称"本应用"）是一款开源的 Android SSH 服务器应用，用于在 Android 设备上运行 SSH 服务。

开源地址：github.com/JoursBleu/ssh-server

2. 数据收集

本应用不收集任何个人信息。

  • 不收集设备信息（IMEI、设备ID等）
  • 不收集位置信息
  • 不收集通讯录、短信、通话记录
  • 不收集应用使用数据或行为数据
  • 不包含任何第三方分析工具（如 Firebase Analytics、友盟等）
  • 不包含任何广告 SDK

3. 网络访问

本应用需要网络权限，仅用于以下目的：

  • 在本地网络（局域网）上监听 SSH 连接请求
  • 为 SSH 端口转发功能提供网络通信

本应用不会主动向任何外部服务器发起网络请求，不上传任何数据。

4. 本地存储

本应用在设备本地存储以下数据：

  • SSH 主机密钥：首次启动时生成的 RSA 密钥对，存储在应用私有目录
  • 授权公钥：用户手动导入的 SSH 公钥（authorized_keys）
  • 用户设置：端口号、认证方式等配置信息

以上数据均存储在应用私有目录中，卸载应用时自动删除。

5. 权限说明

  • 网络访问（INTERNET）：SSH 服务器监听和端口转发所需
  • 前台服务（FOREGROUND_SERVICE）：保持 SSH 服务在后台稳定运行
  • 开机自启（RECEIVE_BOOT_COMPLETED）：可选功能，用于开机自动启动 SSH 服务

6. 数据安全

本应用采用以下安全措施：

  • SSH 连接使用标准 SSH2 协议加密
  • 支持 RSA 3072 位密钥
  • 密钥文件存储在 Android 应用沙箱中，其他应用无法访问
  • 使用 BouncyCastle 密码学库提供加密支持

7. 第三方服务

本应用不集成任何第三方服务、SDK 或数据追踪工具。

8. 儿童隐私

本应用不面向 13 岁以下儿童，不会故意收集儿童的个人信息。

9. 隐私政策变更

如本隐私政策有更新，将在本页面发布。建议您定期查看。

10. 联系方式

如有任何隐私相关问题，请通过以下方式联系：

  • GitHub Issues：github.com/JoursBleu/ssh-server/issues

Privacy Policy

Last updated: March 14, 2026

TL;DR: SSH Server does not collect, store, or transmit any user data. The app runs entirely locally, does not connect to the internet on its own, and contains no analytics or advertising SDKs.

1. About the App

SSH Server ("the App") is an open-source Android application that runs an SSH server on your Android device.

Source code: github.com/JoursBleu/ssh-server

2. Data Collection

The App does not collect any personal information.

  • No device identifiers (IMEI, device ID, etc.)
  • No location data
  • No contacts, messages, or call logs
  • No usage or behavior analytics
  • No third-party analytics tools (Firebase, etc.)
  • No advertising SDKs

3. Network Access

The App requires network permission solely for:

  • Listening for incoming SSH connections on the local network
  • Providing SSH port forwarding functionality

The App never initiates outbound connections to any external server and does not upload any data.

4. Local Storage

The App stores the following data locally on your device:

  • SSH host keys: RSA key pair generated on first launch, stored in the app's private directory
  • Authorized keys: SSH public keys manually imported by the user
  • User settings: Port number, authentication preferences, etc.

All data is stored in the app's private directory and is automatically deleted when the app is uninstalled.

5. Permissions

  • INTERNET: Required for SSH server listening and port forwarding
  • FOREGROUND_SERVICE: Keeps the SSH service running reliably in the background
  • RECEIVE_BOOT_COMPLETED: Optional, for auto-starting SSH service on device boot

6. Data Security

  • All SSH connections are encrypted using the standard SSH2 protocol
  • RSA 3072-bit keys are used
  • Key files are stored in the Android app sandbox, inaccessible to other apps
  • Cryptographic operations are provided by BouncyCastle

7. Third-Party Services

The App does not integrate any third-party services, SDKs, or tracking tools.

8. Children's Privacy

The App is not directed at children under 13 and does not knowingly collect personal information from children.

9. Changes to This Policy

Any updates to this privacy policy will be posted on this page.

10. Contact

For privacy-related questions, please contact us via:

  • GitHub Issues: github.com/JoursBleu/ssh-server/issues
